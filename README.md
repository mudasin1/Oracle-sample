# Oracle-sample

## Pro*COBOL copy/precompile flow

This repo includes scripts to generate COBOL copies from `.pco` files without renaming originals, and to run Oracle Pro*COBOL precompilation.

### 1) Configure `procob`

```bash
./scripts/configure_procob_env.sh
```

If your Oracle home is not auto-detected:

```bash
./scripts/configure_procob_env.sh --oracle-home /path/to/oracle/home
```

This creates `.procob.env` in the repo root.

### 2) Run copy + precompile

```bash
source .procob.env
./scripts/procob_flow.sh
```

Outputs:
- Copy output: `*.cbl` (exact copy of source `.pco`)
- Precompile output: `*.procob.cbl` (generated by `procob`)

### Useful options

```bash
# only make .cbl copies
./scripts/procob_flow.sh --copy-only

# only run precompile
./scripts/procob_flow.sh --precompile-only

# do not overwrite existing generated files
./scripts/procob_flow.sh --no-overwrite
```
